Mthds CLI Setup

Automate your workflows by installing the Python Micro-Sprint Agent.

1. Install the Package

Ensure you have Python 3.9+ and Git installed. Clone the repository and install it globally:

git clone https://github.com/mthds/mthds-cli.git cd mthds-cli pip install -e .

2. Authenticate

Copy one of your tokens from below to log in:

mthds login <your-token>

3. Start the Micro-Sprint Engine

Setup an AI model in ~/.mthds/ai_config.json, then run your board (e.g., ID 12):

mthds run 12
